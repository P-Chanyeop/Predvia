# PREDVIA - 구매대행 프로그램

"구매대행, 반자동 그 이상" - 최고의 디테일로 완성된 스마트 솔루션 프로그램

## 프로젝트 개요

PREDVIA는 구매대행 업무를 효율적으로 관리할 수 있는 데스크톱 애플리케이션입니다. Avalonia UI 프레임워크를 사용하여 C#으로 개발되었으며, 크로스 플랫폼을 지원합니다.

## 주요 기능 (구현 완료)

- **인증 시스템**: API 키 기반 소프트웨어 인증
- **테마 시스템**: 다크모드/라이트모드 테마 전환
- **소싱 페이지**: 
  - 필터 기능 (가격 범위 설정)
  - 수동 소싱 (URL 입력 기반)
  - 자동 소싱 (소싱제로, 메인상품 자동찾기)
  - 엑셀 수출 기능
  - 데이터 유무에 따른 동적 화면 전환
- **UI/UX**: 
  - 탭 기반 메뉴 시스템
  - 1920x1080 해상도 최적화
  - 중앙 정렬 및 반응형 디자인
  - 오렌지 테마 색상 통일
- **외부 연동**: 한국어 명언 API 연동

## 기술 스택

- 언어: C#
- UI 프레임워크: Avalonia UI
- 대상 플랫폼: Windows, macOS, Linux
- 외부 API: 한국어 명언 API

## 설치 및 실행 방법

```bash
# 프로젝트 복제
git clone https://github.com/your-username/Gumaedaehang.git

# 디렉토리 이동
cd Gumaedaehang

# 의존성 설치
dotnet restore

# 애플리케이션 실행
dotnet run
```
