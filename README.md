# PREDVIA - 구매대행 프로그램

"구매대행, 반자동 그 이상" - 최고의 디테일로 완성된 스마트 솔루션 프로그램

## 프로젝트 개요

PREDVIA는 구매대행 업무를 효율적으로 관리할 수 있는 데스크톱 애플리케이션입니다. Avalonia UI 프레임워크를 사용하여 C#으로 개발되었으며, 크로스 플랫폼을 지원합니다.

## 개발 진행 상황

### 2025-07-16
- 프로젝트 초기 설정 완료
- 메인 화면 UI 구현 (라이트 테마, 비로그인 상태)
- 1920x1080 해상도에 맞게 디자인 최적화

## 주요 기능 (계획)

- 사용자 인증 (로그인/회원가입)
- 다크모드/라이트모드 테마 전환
- 구매대행 상품 관리
- 주문 처리 및 추적
- 재고 관리
- 통계 및 보고서

## 기술 스택

- 언어: C#
- UI 프레임워크: Avalonia UI
- 대상 플랫폼: Windows, macOS, Linux

## 설치 및 실행 방법

```bash
# 프로젝트 복제
git clone https://github.com/your-username/Gumaedaehang.git

# 디렉토리 이동
cd Gumaedaehang

# 의존성 설치
dotnet restore

# 애플리케이션 실행
dotnet run
```
